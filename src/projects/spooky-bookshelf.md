# {{ title }}

The Spooky Bookshelf (TSB) is my homage to the indie web, going back to a time when websites were built with tons of colors, bad design decisions, and most of all... FUN.

TSB uses the Build Awesome (11ty) static site generator, <a href="https://neocities.com" target="_blank" rel="noopener noreferrer">Neocities</a> for hosting (I'm a supporter), and uses a GitHub workflow to make it post from GitHub to Neocities.
